# 🔴 სესია 5: კალმარის თამაში (Control Structures)

ნამდვილი თამაში იწყება. პროგრამირებაში ლოგიკა წყვეტს ყველაფერს: ერთი არასწორი პირობა და თქვენი კოდი თამაშიდან გამოთიშულია! თქვენი მიზანია დაწეროთ უსაფრთხოების ლოგიკა ეკრანზე მოცემული პერსონაჟისთვის (PLAYER).

## 📜 თამაშის წესები:

ფუნქცია იღებს ორ პარამეტრს:

1. `light` - შუქნიშნის ფერი (`"Red"` ან `"Green"`).
2. `timeLeft` - ფინიშამდე დარჩენილი დრო წამებში (იწყება 60 წამიდან).

## ⚔️ მისია:

გახსენით `index.html` ფაილი და რედაქტორში დაასრულეთ ლოგიკა, რომელიც დააბრუნებს (`return`) პერსონაჟის მოქმედებას:

1. **თუ შუქი არის `"Red"`:** მოთამაშე აუცილებლად უნდა გაჩერდეს! დააბრუნეთ `"stop"`.
2. **თუ შუქი არის `"Green"` და დარჩენილი დრო არის 10 წამზე მეტი (`> 10`):** იარეთ ჩვეულებრივ. დააბრუნეთ `"move"`.
3. **თუ შუქი არის `"Green"` და დრო 10 წამია ან ნაკლები (`<= 10`):** დრო იწურება! თქვენ უნდა გაიქცეთ ორმაგი სისწრაფით. დააბრუნეთ `"sprint"`.

_გაფრთხილება: თუ თქვენი მოთამაშე უბრალოდ "move"-ით ივლის მთელი თამაში, დრო ამოეწურება და თოჯინა გამოგთიშავთ! ბოლო წამებში სპრინტი სასიცოცხლოდ მნიშვნელოვანია._

## 📤 დავალების ჩაბარება:

1. შეავსეთ კოდი და გაუშვით თამაში (▶ გაშვება).
2. დაელოდეთ სანამ თქვენი პერსონაჟი მიაღწევს ფინიშს.
3. გადაუღეთ ეკრანს სქრინშოთი გამარჯვების მომენტში (ისე, რომ დაწერილი ლოგიკაც ჩანდეს) და ატვირთეთ სისტემაში.
